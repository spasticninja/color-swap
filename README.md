# Color Swap
A react color tile swap game! Inspired by Color Puzzle and I Love Hue.

## Getting Started
1. Pull down project. 
2. Run `npm install`
3. Start app with `npm run dev`. 
4. Go to `http://localhost:1234` in your browser to view the app.
